# Менеджер Загруженности Парковки

Простое веб-приложение на Python для анализа загруженности парковки с использованием YOLOv8.

## Возможности
- Загрузка изображений парковки
- Автоматическое обнаружение автомобилей с помощью YOLOv8
- Статистика загруженности (свободные места, % занятости)
- История запросов сохраняется в JSON
- Экспорт истории в Excel

## Установка

1. Установите Python 3.9+
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите приложение:
   ```bash
   streamlit run app.py
   ```

## Использование
1. Введите общее количество парковочных мест
2. Загрузите изображение парковки
3. Нажмите "Запустить обработку"
4. Просмотрите результаты и скачайте отчеты
